# 🔐 Basic Access Control
My first Python project focused on authentication logic and user input handling.

##  Features
- **Identity Verification:** Checks for a valid ID before asking for a password.
- **Data Sanitization:** Uses `.lower()` to prevent errors with capital letters.
- **Nested Logic:** Implements nested `if/else` statements for multi-layered security.

##  What I Learned
- How to capture and store user input.
- Basic flow control and conditional branching.
- How to use if/else.
