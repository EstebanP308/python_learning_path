### 2. 🏦 [Loan Approval Simulator](./Loan_Simulator.py)
*A financial risk assessment system based on real-world banking standards.*

- **How ​​it Works:** The program requests key financial data to calculate the **DTI (Debt-to-Income ratio)**. It cross-references this percentage with the user's **Credit Score** to determine their creditworthiness.
- **Decision Logic:**
    - Determines whether the user is "overextended" (DTI > 50%).
    - Classifies credit into 5 levels (ranging from *Poor* to *Exceptional*).
    - Applies approval logic based on a combination of high income and credit stability.
- **CS Concepts:** Use of arithmetic operators for financial formulas, chained `if/elif/else` statements, and nested logic.
- **Status:** ✅ Completed
