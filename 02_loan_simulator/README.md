# 🏦 Loan Approval Simulator
A financial tool that evaluates loan eligibility based on real-world banking standards (FICO & DTI).

## 📊 Key Features
- **DTI Calculation:** Automatically calculates the Debt-to-Income ratio.
- **Risk Assessment:** Categorizes credit scores from Poor to Exceptional.
- **Conditional Logic:** Uses complex and/or operators to determine final approval.

## 🧮 The Formula
DTI = (Total Monthly Debts / Gross Monthly Income) * 100

## 🛠️ Concepts Applied
- Arithmetic operators and float/int conversion.
- Advanced if/elif/else structures.
- Logic-based decision making for risk management.
